# math-practice-site
A site to practice math problems, currently addition, subtraction, multiplication, &amp; division.  The site includes a  drawing area and some settings to adjust problem difficulty.

## How to use
By default, your operator(addition, subtraction, multiplication, division) is set to "Random" and the number of digits is set to single digits.

You can modify both the operator and the amount of digits a number can have through the settings menu. Click on "Settings" to view options.

Once everything is set how you prefer, you will see a problem displayed to solve. Use the provided workspace as needed and when you find the solution, enter it into the answer box and check your answer.

## Application Setup
Double click the index.html file to run locally in your browser.

## Dependencies
The application has several dependencies that are loaded via CDNs(Content Delivery Network)
- jQuery v3.4.1
- Signature Pad v2.3.2
